Bolalar uchun mo'ljallangan, zamonaviy va interaktiv **Karra Jadvali** o'yini. Ushbu loyiha maxsus **Telegram Web App (Mini App)** sifatida ishlab chiqilgan bo'lib, Telegram interfeysining ichida to'liq moslashuvchan rejimda ishlaydi.

---

## 🇺🇿 O'zbekcha Tavsif

### ✨ Xususiyatlari
- **Telegram Integratsiyasi**: Foydalanuvchi ismini avtomatik aniqlash (Greeting badge) va Telegram mavzusiga (Theme Colors) mos ravishda interfeys ranglarining avtomatik moslashishi.
- **Haptic Feedback (Vibratsiya)**: Telegram Web App SDK orqali javoblarning to'g'riligi yoki xatoligiga qarab foydalanuvchiga sensor aloqa (vibratsiya) berish tizimi.
- **Premium Dizayn (Glassmorphic UI)**:
  - Zamonaviy *Outfit* shrifti.
  - Silliq va chiroyli gradientlar, hover effektlari va mikromodern animatsiyalar.
  - Dark mode uchun moslashtirilgan shaffof oyna effekti.
- **O'yin mexanikasi**:
  - Har safar tasodifiy generatsiya qilinadigan 10 ta savol.
  - Variantlar orasidan to'g'ri javobni tanlash imkoniyati.
  - Natijani hisoblash va o'yinni qaytadan boshlash tugmasi.

## 🇬🇧 English Description

### ✨ Features
- **Telegram Mini App Integration**: Automatically welcomes the user using their Telegram profile details and dynamically adapts the UI colors to match the active Telegram theme.
- **Haptic Feedback**: Utilizes Telegram Web App SDK's Haptic API to trigger tactile feedback (vibrations) on mobile devices for correct, incorrect, and restart actions.
- **Premium Glassmorphic UI**:
  - Uses modern typography (*Outfit* Google Font).
  - Vibrant gradients, smooth hover interactions, and micro-animations.
  - High-performance, hand-crafted Vanilla CSS (no bulky frameworks).
- **Game Mechanics**:
  - Generates 10 random multiplication questions dynamically.
  - Presents 3 interactive options for each equation.
  - Real-time score calculation and instant feedback.
